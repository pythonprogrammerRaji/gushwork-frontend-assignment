Project: Gushwork Assignment - HDPE Piping Solutions Page
Author : Rajeshwari

Project Overview:
A clean, interactive frontend implementation of a manufacturing sector landing page. Developed using vanilla web technologies to ensure lightweight performance, featuring smooth CSS transitions, interactive product galleries, and a fully responsive contact interface.

TECHNICAL OVERVIEW:
- HTML5: Used semantic tags (header, section, footer) for SEO and accessibility.
- CSS3: Built entirely with Flexbox. No frameworks (Bootstrap/Tailwind) used.
- JavaScript: Vanilla JS used for the sticky header, carousel, and zoom effects.

Project Structure
├── index.html
├── styles.css
├── script.js
└── README.md


KEY FEATURES:
- Sticky Header: Activates after scrolling past the first fold.
- Image Zoom: 1.5x scale transition on hover for product detail.
- Responsive Rows: Horizontal scrolling implemented for mobile cards to prevent layout breakage.
